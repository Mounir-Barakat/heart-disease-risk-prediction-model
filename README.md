# Heart Disease Prediction Model
"ML model predicting heart disease using Random Forest"

## Overview
A machine learning model that predicts 
heart disease using Random Forest algorithm

## Results
- Accuracy: 83.61%
- Dataset: 303 patients
- Features: 13 medical measurements

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

## How To Run
1. Open in Google Colab
2. Run all cells
3. Use prediction tool
